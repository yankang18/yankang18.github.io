еҸ‘иЎЁдәҺ 2024-03-15 жӣҙж–°дәҺ 2026-05-18 еҲҶзұ»дәҺ жҠҖжңҜ

Hexo жҳҜдёҖдёӘеҝ«йҖҹгҖҒз®ҖжҙҒдё”й«ҳж•Ҳзҡ„еҚҡе®ўжЎҶжһ¶гҖӮжң¬ж–Үе°Ҷд»Ӣз»ҚеҰӮдҪ•дҪҝз”Ё Hexo жҗӯе»әдёҖдёӘзІҫзҫҺзҡ„дёӘдәәеҚҡе®ўгҖӮ

дёәд»Җд№ҲйҖүжӢ© Hexoпјҹ

  • жһҒйҖҹз”ҹжҲҗпјҡеҹәдәҺ Node.jsпјҢж•°зҷҫзҜҮж–Үз« еҸҜеңЁж•°з§’еҶ…з”ҹжҲҗ
  • Markdown ж”ҜжҢҒпјҡдҪҝз”Ё Markdown зј–еҶҷж–Үз« пјҢдё“жіЁеҶ…е®№жң¬иә«
  • дё»йўҳдё°еҜҢпјҡжӢҘжңүеӨ§йҮҸзІҫзҫҺзҡ„ејҖжәҗдё»йўҳ
  • дёҖй”®йғЁзҪІпјҡж”ҜжҢҒ GitHub PagesгҖҒVercel зӯүеӨҡз§Қе№іеҸ°

2. еҲқе§ӢеҢ–еҚҡе®ў

3. жң¬ең°йў„и§Ҳ

4. ж–°е»әж–Үз« 

зӣ®еҪ•з»“жһ„

иҝӣйҳ¶й…ҚзҪ®

дҪ еҸҜд»ҘйҖҡиҝҮдҝ®ж”№ _config.yml жқҘиҮӘе®ҡд№үз«ҷзӮ№дҝЎжҒҜгҖҒдё»йўҳгҖҒжҸ’д»¶зӯүгҖӮжӣҙеӨҡй…ҚзҪ®иҜ·еҸӮиҖғ Hexo е®ҳж–№ж–ҮжЎЈгҖӮ

жҖ»з»“

Hexo и®©жҗӯе»әеҚҡе®ўеҸҳеҫ—ејӮеёёз®ҖеҚ•гҖӮеҸӘйңҖеҮ жӯҘпјҢдҪ е°ұиғҪжӢҘжңүдёҖдёӘеұһдәҺиҮӘе·ұзҡ„еҚҡе®ўз©әй—ҙпјҢејҖе§Ӣи®°еҪ•е’ҢеҲҶдә«гҖӮ

з®ҖзәҰи®ҫи®Ўзҡ„еҹәжң¬еҺҹеҲҷ

еҸ‘иЎЁдәҺ 2024-02-20 жӣҙж–°дәҺ 2026-05-18 еҲҶзұ»дәҺ и®ҫи®Ў

з®ҖзәҰи®ҫи®ЎпјҲMinimalist DesignпјүжҳҜдёҖз§ҚиҝҪжұӮжһҒиҮҙз®ҖжҙҒзҡ„и®ҫи®ЎйЈҺж јгҖӮе®ғдёҚжҳҜз®ҖеҚ•ең°еҮҸе°‘е…ғзҙ пјҢиҖҢжҳҜжҸҗзӮјзІҫеҚҺпјҢи®©ж ёеҝғеҶ…е®№жӣҙеҠ зӘҒеҮәгҖӮ

ж ёеҝғзҗҶеҝө

вҖңе°‘еҚіжҳҜеӨҡвҖқ вҖ”вҖ” и·Ҝеҫ·з»ҙеёҢВ·еҜҶж–ҜВ·еҮЎеҫ·зҪ—

з®ҖзәҰи®ҫи®Ўзҡ„жң¬иҙЁжҳҜйҖҡиҝҮеҺ»йҷӨеӨҡдҪҷзҡ„иЈ…йҘ°пјҢи®©з”ЁжҲ·иғҪеӨҹдё“жіЁдәҺжңҖйҮҚиҰҒзҡ„еҶ…е®№е’ҢеҠҹиғҪгҖӮ

еӣӣеӨ§еҺҹеҲҷ

1. з•ҷзҷҪпјҲWhitespaceпјү

з•ҷзҷҪдёҚжҳҜз©әзҷҪпјҢиҖҢжҳҜдёҖз§Қдё»еҠЁзҡ„и®ҫи®Ўе…ғзҙ гҖӮйҖӮеҪ“зҡ„з•ҷзҷҪеҸҜд»Ҙпјҡ

  • и®©йЎөйқўе‘јеҗёпјҢеҮҸиҪ»и§Ҷи§үеҺӢеҠӣ
  • еј•еҜјз”ЁжҲ·и§ҶзәҝпјҢзӘҒеҮәйҮҚзӮ№еҶ…е®№
  • иҗҘйҖ й«ҳз«ҜгҖҒдјҳйӣ…зҡ„ж°ӣеӣҙ

2. иүІеҪ©е…ӢеҲ¶

з®ҖзәҰи®ҫи®ЎйҖҡеёёйҮҮз”Ёжңүйҷҗзҡ„иүІеҪ©пјҡ

3. жҺ’зүҲжё…жҷ°

  • йҖүжӢ© 1-2 з§Қеӯ—дҪ“е®¶ж—Ҹ
  • е»әз«Ӣжё…жҷ°зҡ„еӯ—еҸ·еұӮзә§
  • дҝқжҢҒи¶іеӨҹзҡ„иЎҢй«ҳе’Ңеӯ—й—ҙи·қ

4. еҠҹиғҪдјҳе…Ҳ

жҜҸдёҖдёӘе…ғзҙ йғҪеә”иҜҘжңүе…¶еӯҳеңЁзҡ„зҗҶз”ұгҖӮй—®иҮӘе·ұпјҡ

  • иҝҷдёӘе…ғзҙ жҳҜеҗҰеё®еҠ©з”ЁжҲ·зҗҶи§ЈеҶ…е®№пјҹ
  • еҺ»жҺүе®ғдјҡеҪұе“Қз”ЁжҲ·дҪ“йӘҢеҗ—пјҹ
  • жҳҜеҗҰжңүжӣҙз®ҖжҙҒзҡ„иЎЁиҫҫж–№ејҸпјҹ

е®һи·өе»әи®®

  1. е…ҲеҠ жі•пјҢеҗҺеҮҸжі•пјҡе…ҲеҲ—еҮәжүҖжңүйңҖиҰҒзҡ„еҶ…е®№пјҢеҶҚйҖҗжӯҘзІҫз®Җ
  2. д»ҘеҶ…е®№дёәдёӯеҝғпјҡи®ҫи®ЎжңҚеҠЎдәҺеҶ…е®№пјҢиҖҢйқһзӣёеҸҚ
  3. дҝқжҢҒдёҖиҮҙжҖ§пјҡз»ҹдёҖзҡ„и®ҫи®ЎиҜӯиЁҖиғҪйҷҚдҪҺи®ӨзҹҘиҙҹжӢ…
  4. жіЁйҮҚз»ҶиҠӮпјҡз®ҖзәҰи®ҫи®ЎдёӯпјҢжҜҸдёҖдёӘз»ҶиҠӮйғҪдјҡиў«ж”ҫеӨ§е®Ўи§Ҷ

з»“иҜӯ

з®ҖзәҰи®ҫи®ЎжҳҜдёҖз§ҚжҖҒеәҰпјҢжҳҜеҜ№з”ЁжҲ·ж—¶й—ҙе’ҢжіЁж„ҸеҠӣзҡ„е°ҠйҮҚгҖӮеҪ“жҲ‘д»¬еӯҰдјҡеҒҡеҮҸжі•пјҢи®ҫи®Ўзҡ„еҠӣйҮҸеҸҚиҖҢдјҡжӣҙеҠ ејәеӨ§гҖӮ

гҖҠеҺҹеӯҗд№ жғҜгҖӢиҜ»д№Ұз¬”и®°

еҸ‘иЎЁдәҺ 2024-01-10 жӣҙж–°дәҺ 2026-05-18 еҲҶзұ»дәҺ йҳ…иҜ»

и©№е§Ҷж–ҜВ·е…ӢеҲ©е°”зҡ„гҖҠеҺҹеӯҗд№ жғҜгҖӢжҳҜдёҖжң¬е…ідәҺд№ жғҜе…»жҲҗзҡ„з»Ҹе…ёд№ӢдҪңгҖӮд№ҰдёӯжҸҗеҮәдәҶи®ёеӨҡе®һз”Ёзҡ„ж–№жі•и®әпјҢеё®еҠ©жҲ‘д»¬е»әз«ӢеҘҪд№ жғҜгҖҒжҲ’йҷӨеқҸд№ жғҜгҖӮ

ж ёеҝғи§ӮзӮ№

1% зҡ„еӨҚеҲ©ж•Ҳеә”

жҜҸеӨ©иҝӣжӯҘ 1%пјҢдёҖе№ҙеҗҺдҪ дјҡиҝӣжӯҘ 37 еҖҚпјӣжҜҸеӨ©йҖҖжӯҘ 1%пјҢдёҖе№ҙеҗҺдҪ дјҡеҮ д№ҺеҪ’йӣ¶гҖӮ

д№ жғҜжҳҜиҮӘжҲ‘жҸҗй«ҳзҡ„еӨҚеҲ©гҖӮ

д№ жғҜе…»жҲҗзҡ„еӣӣдёӘе®ҡеҫӢ

йҳ¶ж®ө е®ҡеҫӢ ж–№жі•
жҸҗзӨә и®©е®ғжҳҫиҖҢжҳ“и§Ғ зҺҜеўғи®ҫи®ЎгҖҒд№ жғҜеҸ еҠ 
жёҙжңӣ и®©е®ғжңүеҗёеј•еҠӣ е–ңеҘҪз»‘е®ҡгҖҒзҫӨдҪ“еҪ’еұһ
еҸҚеә” и®©е®ғз®Җдҫҝжҳ“иЎҢ еҮҸе°‘йҳ»еҠӣгҖҒдёӨеҲҶй’ҹи§„еҲҷ
еҘ–еҠұ и®©е®ғд»Өдәәж„үжӮҰ еҚіж—¶еҘ–еҠұгҖҒд№ жғҜиҝҪиёӘ

жҲ‘зҡ„е®һи·ө

иҜ»е®Ңиҝҷжң¬д№ҰеҗҺпјҢжҲ‘е°қиҜ•дәҶд»ҘдёӢеҮ дёӘж–№жі•пјҡ

  1. д№ жғҜеҸ еҠ пјҡеңЁзҺ°жңүд№ жғҜеҗҺж·»еҠ ж–°д№ жғҜпјҢжҜ”еҰӮ"ж—©жҷЁжіЎе’–е•ЎеҗҺпјҢеҶҘжғі 5 еҲҶй’ҹ"
  2. зҺҜеўғи®ҫи®ЎпјҡжҠҠиҝҗеҠЁжңҚж”ҫеңЁеәҠеӨҙпјҢйҷҚдҪҺиҝҗеҠЁзҡ„еҗҜеҠЁжҲҗжң¬
  3. дёӨеҲҶй’ҹи§„еҲҷпјҡд»»дҪ•д№ жғҜйғҪеҸҜд»ҘеңЁдёӨеҲҶй’ҹеҶ…ејҖе§ӢпјҢе…Ҳе»әз«ӢеҠЁдҪңпјҢеҶҚдјҳеҢ–ж—¶й•ҝ

з»Ҹе…ёиҜӯеҪ•

  • вҖңдҪ дёҚйңҖиҰҒжҸҗеҚҮзӣ®ж ҮпјҢдҪ еҸӘйңҖиҰҒйҷҚдҪҺж ҮеҮҶгҖӮвҖқ
  • вҖңиә«д»ҪиҪ¬еҸҳжҳҜжңҖејәеӨ§зҡ„ж”№еҸҳж–№ејҸгҖӮдёҚжҳҜжҲ‘жғіжҲ’зғҹпјҢиҖҢжҳҜжҲ‘дёҚжҠҪзғҹгҖӮвҖқ
  • вҖңжҲҗеҠҹзҡ„жңҖеӨ§еЁҒиғҒдёҚжҳҜеӨұиҙҘпјҢиҖҢжҳҜж— иҒҠгҖӮвҖқ

жҖ»з»“

гҖҠеҺҹеӯҗд№ жғҜгҖӢз»ҷжҲ‘зҡ„жңҖеӨ§еҗҜеҸ‘жҳҜпјҡе…іжіЁзі»з»ҹпјҢиҖҢйқһзӣ®ж ҮгҖӮеҪ“дҪ е»әз«ӢдәҶжӯЈзЎ®зҡ„ж—Ҙеёёзі»з»ҹпјҢжғіиҰҒзҡ„з»“жһңиҮӘз„¶дјҡйҡҸд№ӢиҖҢжқҘгҖӮ
